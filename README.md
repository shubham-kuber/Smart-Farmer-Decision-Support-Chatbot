This project is a Python-based chatbot designed to help farmers make better decisions related to:

- Crop cultivation practices
- Quantity-based selling strategies
- Market channel selection
- Value addition awareness (grading, packing, storage)

## Crops Covered
- Tomato
- Onion
- Potato
- Banana
- Grapes
- Pomegranate

## Key Focus
This is a logic-based decision support system developed as a learning-phase project, focusing on real-world agricultural problems rather than paid APIs or live data.

## Tech Stack
- Python
- Jupyter Notebook / Python Script

## Future Scope
- API integration for live market prices
- Web interface
- Cost-profit analysis
